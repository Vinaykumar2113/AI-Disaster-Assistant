# AI Disaster Response Assistant

An AI-powered web application that helps citizens and emergency responders during natural disasters.

## Problem

During disasters, people struggle to find reliable information, nearby shelters, emergency contacts, and safety guidance quickly.

## Solution

The AI Disaster Response Assistant provides:
- AI-powered emergency guidance
- Disaster reporting
- Nearby shelter recommendations
- Emergency alerts
- Incident tracking
- Multi-role dashboards

## Tech Stack

Frontend: React
Backend: FastAPI
Database: PostgreSQL
AI: Python + LLM
Maps: Leaflet/OpenStreetMap
Authentication: JWT
